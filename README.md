# hello_gleam

A simple [elli][1]-based hello world app, written in [Gleam][2].

## Quick start

```sh
# Build the project
rebar3 compile

# Run the eunit tests
rebar3 eunit

# Run the Erlang REPL
rebar3 shell

# Build the release
rebar3 release
```
